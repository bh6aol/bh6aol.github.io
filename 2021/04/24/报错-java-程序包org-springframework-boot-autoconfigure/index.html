报错:java:程序包org.springframework.boot.autoconfigure

0、问题

maven 包管理好像也并不是很给力啊，今天写springboot时，引了几个jar包，maven就抽风了。突然就报如下的错误：

这也太夸张了吧？这个报错！

尝试

  • 重启IDEA（无效）
  • clean maven（无效）

1、解决

  • 右击项目选择Maven -》点击Reload project

    如果不行的话就使用下面的命令，在当前项目的控制台执行更新不完整依赖命令
