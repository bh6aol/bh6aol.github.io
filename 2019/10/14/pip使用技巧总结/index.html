1，使用pip安装新的库
pip install 库名

2，列出已经安装的库
pip list

3，列出过期的库
pip list –outdated

4，升级库
pip install –upgrade 库名

5，显示包的详细信息
pip show 库名

6，安装指定版本号的包
pip install -v 库名==版本

7，卸载
pip uninstall 库名

8 ，查找包
pip search package
